# Admin Control Center - Portfolio Management System

## Project Overview

This Admin Control Center is a comprehensive management system designed for Deepak Kumar's professional portfolio. It provides a user-friendly interface to manage every aspect of the portfolio without requiring any coding knowledge. The system allows for real-time updates to content, media, pricing, and lead management.

## Features

### 🛠️ Hero & Profile Management
- Dynamic text updates for headlines and sub-headlines
- Service tags management for rotating professional titles
- Profile image upload with automatic web optimization

### 📁 Portfolio & Media Gallery
- Project categorization (Development, Application, Photoshop, etc.)
- Video embedding support for Reels, Shorts, and long-form content
- Metric tracking for project popularity

### 📝 Resume & Skills Control
- Experience timeline management
- Skill rating adjustments via sliders
- Education credential updates

### 💰 Pricing & Services
- Tier management for Static, Standard, and Premium packages
- Feature checklist updates for each package
- Recommendation toggling for highlighted packages

### ✍️ Blog & Testimonials
- Full-featured CMS for blog posts
- Testimonial management system
- Client logo management for "Awesome Clients" section

### 📧 Lead Management System
- Inquiry dashboard with centralized table view
- Status tracking (New, In-Progress, Completed)
- Contact export to CSV/Excel

## Technology Stack. http://localhost/personalPortfolio/Admin/login.php

- **CSS3** - For responsive styling and animations

## Installation & Setup

1. Place all files in your web server directory
2. Access the admin panel via `/Admin/login.html`
3. Use the default credentials (to be configured) or contact administrator for access
4. After successful login, you'll be redirected to the main dashboard
